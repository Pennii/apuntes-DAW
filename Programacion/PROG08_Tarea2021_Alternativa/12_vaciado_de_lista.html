1.2.- Vaciado de lista

Savana Price (Pixabay License)

Implementa un programa en Java donde:

  1. se rellene una estructura de tipo lista con doce números enteros aleatorios entre 1 y 5 (ambos incluidos) y se muestre su contenido por pantalla;
  2. a continuación se observará el contenido del primer elemento de la lista y se eliminarán de ella todos los elementos que tengan ese valor, volviéndose a mostrar cómo ha quedado la lista tras el borrado;
  3. tras esto se observará nuevamente el valor del primer elemento de la lista (que necesariamente será otro diferente al del paso anterior, pues todas las posibles apariciones de ese valor habrán sido eliminadas) y se borrarán también todos los elementos que tengan ese valor, mostrando de nuevo el contenido de la lista;
  4. se siga procediendo de ese modo hasta que la lista quede completamente vacía.

Ejemplo de ejecución

Un ejemplo de ejecución podría ser el siguiente:

Retroalimentación

VACIADO DE LISTA
----------------
Contenido inicial de la lista: [5, 3, 5, 2, 1, 1, 4, 3, 3, 4, 1, 4]
Procedemos a ir vaciando su contenido.

Eliminamos todas las apariciones del primer elemento: 5
Lista resultante: [3, 2, 1, 1, 4, 3, 3, 4, 1, 4]

Eliminamos todas las apariciones del primer elemento: 3
Lista resultante: [2, 1, 1, 4, 4, 1, 4]

Eliminamos todas las apariciones del primer elemento: 2
Lista resultante: [1, 1, 4, 4, 1, 4]

Eliminamos todas las apariciones del primer elemento: 1
Lista resultante: [4, 4, 4]

Eliminamos todas las apariciones del primer elemento: 4
Lista resultante: []

Recomendación

Recuerda que si no se pide ningún formato específico de salida para mostrar el contenido de una estructura de datos, siempre puedes utilizar su método toString.

1.4.- Asignación de turnos (I)

200Degrees  (Pixabay License)

Dado el siguiente array con los nombres de las personas que trabajan en sistema de turnos:

String[] arrayPersonas = {"Juan", "Luisa", "Clara", "Rogelio", "Francisco"};

hay que implementar un programa en Java que genere y rellene una estructura de tipo Map llamada mapTurnosPorFecha donde:

  • las claves serán fechas (objetos de tipo LocalDate);
  • los elementos serán listas que contengan nombres de personas (objetos de tipo List<String>).

Una vez declarada la estructura Map se rellenará con siete elementos donde:

  • las claves serán las fechas desde el 1 de marzo al 7 de marzo de 2021 (objetos de tipo LocalDate);
  • los elementos serán listas vacías, aún sin contener nombres de personas.

A partir de aquí, se irán rellenando los conjuntos de personas correspondientes a cada fecha según el siguiente criterio:

  1. cada fecha tendrá asociada tres personas para trabajar ese día. Comenzaremos asignando a la primera persona del array y continuaremos de manera secuencial siguiendo el orden de los nombres en el array. Cuando se vayan a superar los límites del array se volverá a la primera posición (recuerda la utilidad del operador módulo % para estos casos).
  2. la primera persona asignada en una fecha no será asignada para el día siguiente, de manera que de un día para otro coincidirán la segunda y tercera persona del día anterior, que serán ahora primera y segunda, y habrá una tercera persona nueva.

Esto significa que si para el 1 de marzo habría que asignar el trabajo a  Juan, Luisa, y Clara, para el día siguiente (2 de marzo) las personas involucradas tendrían que ser Luisa, Clara y Rogelio, para el tercer día (3 de marzo) tendríamos a Clara, Rogelio y Francisco, y así sucesivamente.

El programa, a la vez que va rellenando la estructura de datos, debe ir indicando por pantalla cómo se va asignando a cada fecha cada uno de las personas involucradas, pudiendo quedar algo así:

Una vez realizadas todas las asignaciones, deberá recorrerse el contenido del mapa de turnos y mostrar, para cada clave (objeto de tipo fecha), el conjunto de personas asociadas a ella usando el siguiente formato:

Fecha dd/mm/aaaa: [persona_1, persona_2, persona_3]

En nuestro caso concreto, el resultado debería quedar así:

Contenido final del mapa de turnos:
Fecha 01/03/2021: [Juan, Luisa, Clara]
Fecha 02/03/2021: [Luisa, Clara, Rogelio]
Fecha 03/03/2021: [Clara, Rogelio, Francisco]
Fecha 04/03/2021: [Rogelio, Francisco, Juan]
Fecha 05/03/2021: [Francisco, Juan, Luisa]
Fecha 06/03/2021: [Juan, Luisa, Clara]
Fecha 07/03/2021: [Luisa, Clara, Rogelio]

Recomendación

Recuerda que diseñar formatos de fecha y hora puedes crear un objeto de la clase DateTimeFormatter mediante el método estático ofPattern. Para usar ese formato tendrás posteriormente que usar el método format sobre el objeto LocalDate que quieras mostrar por pantalla.

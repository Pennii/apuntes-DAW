1.5.- Asignación de turnos (II)

Clker-Free-Vector-Images (Pixabay License)

Partiendo del resultado obtenido por el ejercicio anterior, implementa un programa en Java que explore la estructura de datos que contiene la asignación de turnos organizados por fechas y genere una nueva estructura de datos que contenga la asignación de turnos organizados por personas. Es decir, que se rellene una estructura de tipo Map, en este caso llamada mapTurnosPorPersona, donde:

  • las claves sean los nombres de las personas (objetos de tipo String);
  • los elementos serán listas de fechas (objetos de tipo List<LocalDate>).

Para realizar esta segunda parte, el programa no deberá usar los elementos del array original de nombres, sino que tendrá que obtenerlos a partir de la estructura mapTurnosPorFecha generada por el programa anterior de manera que según la vaya explorando se vayan encontrando nombres de personas que serán añadidos como claves del nuevo mapTurnosPorPersona.

Por tanto, los pasos que habrían de darse serían:

  1. realizar todas las acciones del ejercicio anterior para generar la estructura mapTurnosPorFecha, aunque sin mostrar las asignaciones paso a paso, únicamente el contenido final de la estructura;
  2. llevar a cabo la exploración elemento a elemento de todos los contenidos de ese Map e ir generando una nueva entrada en el nuevo Map mapTurnosPorPersona cada vez que aparezca un nuevo nombre de persona e indicarlo por pantalla;
  3. para cada aparición de una persona debe añadirse, a la lista asociada a esa persona, la fecha correspondiente e indicarlo por pantalla;
  4. una vez finalizada toda la exploración, se recorrerá la nueva estructura mapTurnosPorPersona y se mostrará su contenido de una manera similar a como se hizo con la estructura mapTurnosPorFecha.
